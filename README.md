# Shinra — Centre de contrôle

Poste de pilotage de l'agence Shinra (Bot Lola, ClipFusion Shinra, Shinra Finance).
Jumeau vert du centre de contrôle Noctra, totalement indépendant.

- `HUB_URL` (en haut de `index.html`) — cerveau **Shinra** (stats-tracker Shinra sur
  Railway, `/api/hub`). Vide = mode démo. À activer avec son propre `HUB_TOKEN`
  (indépendant du Noctra pour cloisonner les données agence / perso).
